02. 갈림길

준비하지 않으면, AI가 당신의 기회를 가져갑니다.

AI 광고 시장은 이미 시작됐습니다. 지금까지의 작업 방식으로는 이 속도를 따라갈 수 없습니다. 하지만 같은 변화가, 준비된 사람에게는 새로운 수익 채널이 됩니다.

준비하지 않으면
잃는 것
  • ✕AI가 만든 광고가 내 기회를 가져갑니다. 가상 모델·생성 이미지가 빠르고 저렴하게 광고 시장을 채웁니다.
  • ✕매체별 소재 경쟁에서 밀려납니다. 캠페인당 30~100개 소재 요구 → 촬영 기반으로는 단가·속도 모두 불리.
  • ✕이미지가 어디 쓰이는지 통제 못 합니다. 딥페이크·무단 사용·합성 광고에 사후 대응만 가능.
  • ✕한 번 받은 광고는 한 번으로 끝. 다음 광고는 처음부터 다시 — 모델·소속사 모두 매번 같은 부담.
  • ✕새 광고주와의 연결을 놓칩니다. 발견·검토·계약 흐름이 흩어져 있어 기회의 크기·다양성 모두 줄어듭니다.
지금 준비하면
얻는 것
  • ✓초상권이 반복 수익 구조가 됩니다. 한 번의 디지털 DNA 캡처 → 다수의 광고주·다수의 캠페인에서 활용 → 매월 정산.
  • ✓촬영 없이도 광고 기회가 계속 열립니다. AI 제작 파이프라인이 필요한 소재를 24시간 내 만들어 냅니다.
  • ✓이미지를 명시적으로 통제합니다. 금지 업종 사전 차단 + 모든 광고 양측 승인 필수 + 자동 만료.
  • ✓업종 독점으로 경쟁사 사용을 막습니다. 16개 업종 대분류 기준, 계약 즉시 자동 발동.
  • ✓새 광고주를 CLYNK가 매칭합니다. 적합도·예산·업종 가용성 기반 추천 → 협의 단계가 단축됩니다.
03. CLYNK

한 번의 준비, 반복되는 기회.

CLYNK는 광고 에이전시·대행사·제작사 역할을 하나의 플랫폼으로 통합합니다. K-셀럽의 디지털 자산(KDDC DB)을 사전 확보하여 양측 컨펌 즉시 AI 제작에 착수하고, 16개 업종 대분류 기준 독점을 자동 보호합니다. 모델은 결정만 합니다 — 나머지 흐름은 CLYNK가 잇습니다.

04. Digital DNA

한 번의 세션, 무한한 확장.

Digital DNA 촬영은 한 번이면 됩니다. KDDC 점에서 약 90분의 세션으로 광고 활용의 기준을 만듭니다. 그 이후의 광고는 모두 AI로 무한 확장. 모델은 다시 촬영장에 가지 않아도 됩니다.

각 단계를 클릭하면 상세 내용을 볼 수 있습니다.

  1. 1

    KDDC 점 방문 · 1회 세션 (약 90분)

    서울 KDDC 점에 방문. 본인 인증, 동의서 작성, 세션 진행. 사전 제공된 촬영 가이드에 따라 준비된 헤어·메이크업·의상으로 촬영.

    평일 가능 소속사 동행 가능
    진행 흐름 (총 약 90분)
    • 도착, 본인 확인 (신분증)
    • 동의서 검토·서명, 사용 범위 안내
    • 메인 세션 (스캔·음성·영상 캡처)
    • 결과 확인, 추가 보완 여부 결정
    준비물

    신분증, 본인 명의 휴대폰. 그 외 모든 장비·세팅·메이크업·헤어·기본 의상은 CLYNK가 표준으로 제공합니다. 본인 트레이드마크 룩이 있다면 추가로 가져오셔도 됩니다.

    동행

    소속사 매니저 또는 법정대리인 동행 가능. 별도 대기 공간 제공. 미성년자는 부모 또는 법정대리인 동의서 + 동행이 필수입니다.

    재방문

    외형 큰 변화 (체형·헤어 스타일·시술 등) 발생 시 계약 기간 내 재촬영 1회 무료. 광고 결과물의 일관성 유지가 목적입니다.

  2. 2

    3D 스캔 · 음성 · 짧은 영상 캡처

    Esper LightCage 기반 상반신·얼굴 3D 스캔, 다양한 표정·각도 사진, 음성 샘플(5분 내외 스크립트), 짧은 동작 영상까지. AI 합성에 필요한 디지털 자산 한 세트를 한 번에 확보합니다. (하반신은 AI 합성 단계에서 자유롭게 생성 가능합니다.)

    Esper LightCage 상반신·얼굴 스캔 음성 샘플 표정 라이브러리
    장비

    Esper LightCage (스몰 사이즈) — 측지 돔 프레임에 다중 카메라와 156개의 MultiFlash 조명을 배치한 photogrammetry 캡처 리그. 상반신·얼굴 360도를 동시에 캡처해 고해상도 3D 메시·텍스처를 한 번에 생성합니다. 하반신은 AI 합성 단계에서 자유롭게 생성되므로 별도 스캔이 필요하지 않습니다.

    캡처 항목
    • 상반신·얼굴 3D 스캔 — Esper LightCage로 360도 메시 + 얼굴·헤어 고해상도 텍스처 (~15분)
    • 표정 라이브러리 — 자연·웃음·진지·집중·놀람 등 12종 (~10분)
    • 상반신 포즈 라이브러리 — 정면·측면·각도별 상체 포즈 등 8종 (~5분)
    • 음성 샘플 — 약 5분 분량 표준 스크립트 + 다양한 톤 변형 (~10분)
    • 짧은 영상 — 자연스러운 상반신 동작 5~10초 클립 다수 (~10분)
    의상

    기본 캐주얼·정장 상의 2종 + 본인 트레이드마크 룩 1종까지 가능. 의상 변경은 1회 허용 (시간 단축을 위함). 하반신·전신 의상, 광고별 의상 합성은 모두 AI 단계에서 처리.

    데이터 처리·보안

    원본 데이터는 캡처 즉시 KDDC 내부 전용 DAS에 저장되며, 외부 네트워크와 분리된 오프라인 환경에서 보호됩니다. 어떤 원본도 온라인 클라우드에 업로드되지 않고, 승인된 내부 권한자만 제한적으로 접근할 수 있습니다. KDDC는 폐쇄형 보안 체계 안에서 원본을 엄격히 관리합니다.

  3. 3

    프로필 등록 · 선호·금지 업종 설정

    CLYNK 모델 프로필 등록. 광고하고 싶은 업종 (선호 5개), 절대 광고하지 않을 업종 (제외) 명시. 이 설정은 광고주 검색 시 자동으로 필터링되어, 원하지 않는 제안은 처음부터 들어오지 않습니다.
    등록 항목
    • 활동명·대표 작품·수상 이력 — 광고주 검색 결과에 노출
    • 이미지 키워드 — 톤·분위기 (예: 신뢰감·청량·세련) — AI 매칭 정확도 향상에 사용
    • 가격 티어 — 신인·신예 / 중견급 / 톱 (소속사와 협의 후 결정)
    • 일정 가용성 — 월 단위 가용/불가 마킹 (선택, 미설정 시 항상 가용)
    업종 설정

    선호 업종 (최대 5개): 16개 대분류 중 선택. 검색 결과 우선 노출.
    금지 업종 (무제한): 해당 업종 광고주는 검색·제안 단계에서 모델을 발견할 수 없습니다. 제안 자체가 도착하지 않으므로 매번 거절할 필요 없음.

    변경·관리

    모든 설정은 언제든 수정 가능. 변경 즉시 매칭 룰에 반영됩니다. 소속사 담당자가 모델을 대신해서 작성·관리도 가능합니다.

    소요 시간

    일반적으로 30분 내 완료. KDDC 세션 종료 후 1주일 내 등록을 권장합니다.

  4. 4

    광고주 매칭 · 제안 · 양측 승인

    광고주가 CLYNK에서 모델을 검색·제안. 모델·소속사 양측 승인 후 계약 성립. 기획안 단계에서 한 번 더 검토 권한이 있어, 이미지에 맞지 않는 광고는 시작 단계에서 멈출 수 있습니다.
    광고주 검색·매칭

    광고주가 CLYNK 플랫폼에서 업종·예산·기간·이미지 키워드 기반으로 모델을 검색합니다. 적합한 모델 발견 시 제안서를 발송합니다. 이때 모델 본인의 사진·연락처는 절대 노출되지 않으며, 등록된 프로필 정보(활동명·대표작·티어·가용 업종)만 보입니다.

    제안서 구성
    • 광고주 정보 — 회사명·브랜드·담당자
    • 캠페인 컨셉·집행 매체 (지면·온라인·OOH 등)
    • 업종·계약 기간 (1·3·6·12개월)
    • 총 금액 = 월 구독료 + 업종 독점 프리미엄
    • 제작 패키지 선택 (Quick~Premium 또는 미선택)
    모델 응답

    수락 / 거절 / 협의 중 선택. 거절 시 사유 (스케줄·브랜드 부적합·금액·업종·기타) 선택 — 향후 매칭 정확도 개선에 사용됩니다. 협의는 금액·기간·조건 수정 제안이 가능합니다.

    소속사 최종 승인

    모델 수락 후 24시간 내 소속사가 검토 → 최종 승인. 승인 직후 광고주 결제 단계 진입. 결제 완료 즉시 업종 독점 자동 발동, KDDC 사용권 활성화됩니다.

  5. 5

    AI 제작 · 검수 · 납품 · 매월 정산

    양측 승인 즉시 AI 제작 시작 (24시간 내). 초안 검토 → 수정 → 최종 납품. 광고 집행 후 매월 정산 (광고주 → CLYNK → 소속사).

    매월 정산 제작비 10% 사용료
    제작 흐름 (양측 승인 후 24시간 내 착수)
    1. 광고주 브리프 접수 → CLYNK 기획안 작성 (1~2일)
    2. 기획안 양측 승인 — 모델·소속사 수정 요청 가능 (이 단계에서도 거절 권한 유지)
    3. AI 제작 — 이미지·영상 생성 (패키지에 따라 3~10일)
    4. 초안 양측 검토 — 수정 횟수 보장 (Quick 1회 / Standard 2회 / Pro 3회 / Premium 5회)
    5. 최종 납품 + CLYNK Verified 배지 표시
    정산 구성 (모델·소속사 수령)
    • 월 구독료 분배분 — 광고주 납부 전액에서 CLYNK 중개 수수료 차감 후, 소속사 ↔ 모델 협의 비율로 분배
    • 업종 독점 프리미엄 — 구독료의 8~30% (Tier 1·2·3에 따라)
    정산 시점·방식

05. Brand Safety

내 얼굴은, 내가 허락한 곳에서만.

이미지는 넓게 활용될 수 있어야 합니다. 하지만 통제 없이 확장되어서는 안 됩니다. CLYNK는 6단계 통제 장치로 소속사·연예인의 권한을 기술과 정책으로 함께 지킵니다.

금지 업종 설정

광고하지 않을 업종을 사전 차단. 해당 업종 제안은 검색·제안 단계에서 차단되어 도착하지 않습니다.

사전 승인 (양측 필수)

모든 제안은 모델 수락 + 소속사 최종 승인 후 진행. CLYNK가 임의로 진행하는 광고는 없습니다.

기획안 검토 (제작 전)

AI 제작 시작 전, 컨셉·카피·구도 기획안 검토 권한. 이미지에 맞지 않으면 이 단계에서 중단합니다.

품질 검수 (납품 전)

계약 범위 명시

업종·기간·매체·지역까지 CLYNK 표준 계약서로 명문화. 범위 외 사용은 분쟁 사유, 3영업일 내 판정.

09. 흐름

가입부터 첫 정산까지.

모델·소속사 입장에서 직접 해야 하는 일은 준비·설정·승인 세 가지입니다. 그 외의 연결·계약·검수·집행·정산은 하나의 흐름으로 이어집니다.

Day 0 — KDDC 문의 5분

웹에서 문의 폼 작성 (이름·소속사·연락처·희망일). 평일 일정 협의 후 확정 알림.

Day 1~7 — Digital DNA 세션 90분

KDDC 점 방문, 본인 인증·동의서, Esper LightCage 상반신·얼굴 3D 스캔과 음성·표정 캡처까지. 한 번이면 끝.

Day 7 — 프로필 등록 + 업종 설정 30분

CLYNK 모델 페이지 활성화. 선호·금지 업종, 가격 티어 (소속사와 협의), 일정 가용성 설정.

Day 7+ — 광고주 매칭 시작 대기

광고주가 CLYNK 검색에서 발견 → 적합도 확인 → 제안 발송. 모델·소속사 알림.

제안 도착 — 검토 + 양측 승인 평균 2~3일

제안 상세 (업종·기간·금액·기획안 컨셉) 확인 → 수락 / 거절 / 협의 → 소속사 최종 승인 → 결제 완료 → 업종 독점 즉시 발동.

제작 단계 — 24h 착수, 3~10일 완성 AI

기획안 검토 (모델·소속사 승인) → AI 제작 → 초안 검토 → 수정 → 최종 납품. 모델은 검토만 합니다.

광고 집행 — CLYNK Verified 집행 중

광고주가 매체 집행. CLYNK Verified 배지로 모델·플랫폼 검증 표시. 갱신 알림 45일 전 자동 발송.

매월 — 정산 입금

광고주 결제 → CLYNK 수령 → 매월 소속사 정산 (구독료 + 독점 프리미엄). 표준 정산서 + 세금계산서 자동 발행.

10. FAQ

자주 묻는 질문.

Digital DNA 촬영 후, 내가 직접 광고 촬영을 해야 하나요?
아니요. Digital DNA 촬영(약 90분 1회)이 끝나면, 이후 모든 광고는 AI로 제작됩니다. 모델은 기획안과 결과물을 검토하는 역할만 수행합니다. 단, 본인이 원하면 일부 광고는 직접 촬영을 병행할 수 있습니다 (개별 협의).
내 이미지가 어디 어떻게 쓰이는지 어떻게 통제하나요?
6단계 안전 장치로 통제합니다. ① 금지 업종 사전 차단 ② 모든 제안에 양측 승인 필수 ③ 기획안 검토 단계에서 거부 권한 ④ 최종 결과물 품질 검수 ⑤ 표준 계약서로 업종·기간·매체·지역 명문화 ⑥ 계약 만료 즉시 사용권 자동 종료. 또한 분쟁 발생 시 3영업일 내 CLYNK 심의 판정.
특정 업종 광고는 절대 안 하고 싶은데, 막을 수 있나요?
네, 가능합니다. 프로필에 "제외 업종"을 명시하면, 해당 업종의 광고주는 검색·제안 단계에서 모델을 발견할 수 없습니다. 제안 자체가 도착하지 않으므로 매번 거절할 필요도 없습니다. 16개 대분류 + 세부 카테고리 단위로 설정 가능합니다.
정산은 언제, 어떻게 받나요?
매월 소속사로 정산됩니다. 정산 항목은 ① 모델 구독료 (광고주 납부 - CLYNK 수수료) ② 업종 독점 프리미엄 (구독료의 8~30%)으로 구성됩니다. 소속사 ↔ 모델 분배 비율은 별도 협의 사항입니다.
계약이 끝나면 내 디지털 자산은 어떻게 되나요?
계약 만료 즉시 사용권이 자동 종료됩니다. 갱신을 원하지 않으면 데이터 삭제 요청도 가능합니다 (요청 후 30일 내 완전 삭제, 백업 포함). 갱신 시에는 다시 양측 승인이 있어야만 사용권이 재발효됩니다. KDDC는 자산을 보관하지만, 사용은 활성 계약 기간에만 가능합니다.
전통적인 광고·드라마 출연 활동을 병행할 수 있나요?
전혀 영향 없습니다. CLYNK는 디지털 자산 기반 광고 사업의 추가 채널입니다. 기존 매니지먼트 활동(드라마·영화·CF·예능·콘서트 등)은 그대로 유지하면서, CLYNK는 추가 수익원으로 작동합니다. 단, 같은 업종에서 직접 광고와 CLYNK 광고가 충돌할 수 있으니 소속사와 사전 조율 필요합니다.
소속사 없이 개인이 가입할 수 있나요?
현재 CLYNK는 KDDC 등록 소속사를 통한 가입을 우선으로 합니다. 개인 가입 (소속사 없음)은 별도 심사 후 가능하며, 정산·법적 보호·표준 계약서 처리 절차가 다소 다릅니다. 무소속 모델도 KDDC 문의 후 협의 진행 가능합니다.
분쟁이 생기면 어떻게 처리되나요?
3영업일 내 CLYNK 심의 판정. 업종 독점 충돌, 계약 범위 외 사용, 결제 지연, 결과물 품질 등의 분쟁은 분쟁 접수 → 양측 의견 수렴 → 3영업일 내 판정 → 결과 통보의 절차로 처리됩니다. 판정 결과에 따라 환불·재제작·계약 조정·법적 절차 안내가 진행됩니다.
Digital DNA 데이터는 어디에 저장되나요? 보안은?
Digital DNA 원본 데이터는 KDDC 내부 전용 DAS에만 저장되며, 외부 네트워크와 분리된 오프라인 환경에서 보호됩니다. 어떤 원본 데이터도 온라인 클라우드에 업로드되지 않으며, 승인된 내부 권한자만 제한적으로 접근할 수 있습니다. KDDC는 원본 데이터가 외부로 노출되지 않도록 설계된 폐쇄형 보안 체계 안에서 이를 엄격히 관리합니다.
