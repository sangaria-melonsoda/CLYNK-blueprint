핵심 강점 — Why CLYNK
경쟁사가
따라올 수 없는 것.

CLYNK의 3가지 핵심 강점은 KDDC와의 독점 구조에서 나옵니다. 플랫폼을 만드는 것으로는 복제할 수 없습니다.

강점 01
KDDC DB 기반
즉시 실행 구조
전체 KDDC 계약 모델의 3D스캔·음성·초상권 데이터 100% 보유. 광고주·연예인 양측 컨펌 완료 즉시 24시간 내 제작 착수 가능. 법적으로 사전 확보된 디지털 자산 기반이라 별도 촬영·소속사 재협의 없음.
경쟁사 한계: 계약 후 소속사 협의 + AI 학습 단계 별도 필요. 제작 가능 여부를 계약 전에 확인할 수 없음.
강점 02
검색·매칭의
패러다임 전환
"누가 팔로워 많은가"가 아닌 "지금 당장 계약·제작·집행까지 완결 가능한 모델"을 찾아주는 구조. 업종 공백 실시간 표시, 즉시 제작 가능 배지, ROI 역산 추천, 브랜드 DNA 기반 AI 매칭까지 제공.
경쟁사 한계: 소셜 지표(팔로워·EGR) 기반 추천. 실제 광고 집행 성과 데이터 연결 없음. 업종 충돌 자동 제외 불가.
강점 03
업종 독점
자동 보호 장치
계약 체결 즉시 16개 업종 대분류 기준 독점 자동 발동. 경쟁사가 동일 모델로 동일 업종 광고를 집행할 수 없음. 3영업일 내 분쟁 판정 체계 포함. 계약 만료 즉시 자동 해제.
경쟁사 한계: 업종 독점 개념 자체 없음. 동일 모델 중복 계약 가능 구조로 광고주 보호 불가.

01
시장 구조
한국 광고 시장은
일본과 다르다

일본은 소속사와 직접 계약해 소재를 공급하는 구조. 한국은 에이전시가 매칭하고 CLYNK를 통해서만 계약과 정산이 이뤄진다.

AI로 인해 한국 광고 시장이 변화하고 있고, 새로운 작품 기회 감소로 연예 매니지먼트 사업도 타격을 받고 있습니다. 대부분의 연예인들이 일거리 감소로 새로운 수입원을 찾는 상황입니다. CLYNK는 이 두 문제를 동시에 풉니다.

왜 지금인가 — 5가지 시장 변화
↑
디지털 광고 비중 확대
온라인·SNS 광고 집행 비중이 전통 매체를 추월. 연예인 활용 디지털 광고 수요 급증.
↑
숏폼·다매체 집행
릴스·틱톡·유튜브 쇼츠 확산으로 동일 모델로 수십 개 소재 필요. AI 제작이 유일한 해법.
↓
캐스팅 프로세스 비효율
레거시 캐스팅은 수일~수주 소요. 중소기업은 시작조차 못하는 진입 장벽 존재.
↓
연예인 수입 구조 붕괴
AI 생성 콘텐츠 확산으로 광고·출연 수입 감소. 소속사들이 새로운 수익 채널을 절실히 필요로 함.
↑
제작 비용 효율 압박
광고주들의 제작 예산 축소 속에서 AI 제작으로 60~80% 비용 절감 가능. 수요가 공급을 앞선다.
모델 공급 확장 로드맵
Phase 1 — 현재
KDDC 전용
KDDC 계약 모델 전원 3D스캔·초상권·음성 데이터 보유. 40명 → 120명 목표.
Phase 2 — 2027
제휴 소속사 확장
KDDC 외 제휴 소속사 파트너십. AI 제작 없는 매칭·중개 전용 모델 풀 운영. 시장 범위 확장.
Phase 3 — 2028+
외부 파트너 온보딩
외부 파트너사 온보딩으로 개방형 플랫폼 전환. K-콘텐츠 글로벌 확장과 연계.

02
포지셔닝
CLYNK가
하는 일

"광고주가 원하는 모델을 찾아주고, 계약을 연결하고, 광고를 만들어서 납품한다."

역할 01 — 핵심
모델 매칭 · 계약 중개
광고주 조건 입력 → KDDC 모델 추천 → 계약 중개. CLYNK가 유일한 계약 통로. 에이전시 역할을 플랫폼이 자동화. 광고주가 원하는 캠페인 브리프를 CLYNK에 전달하면 최적 모델을 연결합니다.
역할 02 — +α
AI 제작 납품
KDDC 3D스캔 데이터 기반 AI 제작. Arkain AI Tool로 직접 제작·납품. 모델 계약자 전용 선택 옵션. 광고주가 원하는 컨셉·브리프를 받아 제작물로 구현합니다.
별도 제작 패키지 선택
광고 기획·전략은 광고주가 준비합니다. CLYNK는 광고주가 원하는 캠페인 방향을 전달받아 최적 모델 매칭과 AI 제작 납품을 수행합니다. 광고주의 브리프를 실행으로 연결하는 것이 CLYNK의 역할입니다.
Pain Point
AI로 일거리가 줄었다
AI 광고 시장 변화로 기존 수입 감소. 새 작품 기회 감소로 매니지먼트 수입도 타격.

03
검색 · 매칭 시스템
완결 가능한 모델을
즉시 찾아준다.

CLYNK 검색은 소셜 지표가 아닌 KDDC DB 기반입니다. "지금 당장 계약·제작·집행까지 완결 가능한가"를 기준으로 추천합니다.

기능 01 — KDDC 전용
"즉시 제작 가능" 배지
3D스캔·음성·초상권 데이터 보유 수준에 따라 검색 결과에 배지 표시. 이미지 전용 / 영상 포함 / 풀세트 가능 레벨로 구분. 납품 예상 기간까지 바로 확인.
기능 02 — 업종 독점 시스템
업종 공백 실시간 표시
모델 프로필에서 현재 계약 중인 업종과 가능한 업종을 실시간 표시. 독점 만료 D-30 알림 신청으로 원하는 모델의 공백 시점 선점 가능.
기능 03 — ROI 시뮬레이션
예산 → ROI 역산 추천
예산 입력 시 모델료 + 제작비 조합 자동 계산. 예상 CTR·방문수까지 시뮬레이션 제공. 구독 기간별 월 단가·AI 제작 패키지 추가 시 ROI 변화 즉시 반영.
기능 04 — 실적 데이터
업종별 실제 광고 성과 추천
소셜 팔로워·EGR이 아닌 CLYNK에서 실제 집행된 광고의 CTR·전환율 기반 추천. "이 모델은 뷰티 업종 평균 CTR 3.2%" 형태로 표시.
기능 05 — 소속사 대시보드 연동
실시간 가용 일정 확인
소속사 파트너 대시보드와 연동해 모델의 실제 가용 여부를 검색 결과에 반영. 조건 맞으면 즉시 계약 확정. "지금 계약 시 X일 내 집행 가능" 즉시 확인.
기능 06 — AI 매칭 고도화
브랜드 DNA 기반 AI 매칭
브랜드 URL·기존 광고물·타겟 고객층 입력 시 브랜드 이미지와 모델 이미지 적합도 AI 분석 후 추천. 단순 프롬프트를 넘어 브랜드 자산 기반 매칭.
경쟁사와의 근본적 차이: 경쟁사 검색은 "소셜 데이터로 누가 영향력 있는가"를 찾는 도구입니다. CLYNK 검색은 KDDC DB를 기반으로 "지금 당장 계약하고 제작까지 완결 가능한 모델이 누구인가"를 찾는 도구입니다. 데이터 소스가 다르기 때문에 복제 불가능합니다.

계약 중개·정산 대행 플랫폼

회계 매출 인식 기준
순액 기준 매출 인식
CLYNK는 총 거래액을 수령하나, 재무상 매출은 중개수수료 및 제작수익만 순액 기준으로 인식합니다. 소속사 정산분은 매출에서 제외합니다.

05
결제 구조
첫 계약은 선납으로,
갱신은 인보이스로.

CLYNK는 계약 체결 즉시 업종 독점이 발동되고 소속사 정산 리스크가 생깁니다. 결제 구조는 이 리스크를 반영한 혼합형으로 설계합니다.

Step 01 — 첫 계약
최소 3개월 선납
계약 체결과 동시에 3개월분 전액 선납. 업종 독점 발동 즉시 현금 확보. 소속사 정산 안정성 보장. 1개월 단위 선택 불가 — 최소 3개월부터.
계약금 + 잔금 구조 아님. 전액 선납.
Step 02 — 갱신 계약
월별 인보이스 허용
첫 계약 완료 후 갱신부터는 월별 세금계산서 발행 + 이체 방식 허용. B2B 표준 방식으로 광고주 편의 제공. 단, 갱신 시에도 업종 독점 프리미엄은 선납.
Step 03 — 부가 옵션
크레딧 차감형
선불 충전 크레딧 구매 후 AI 제작 패키지·추가 업종 독점 등에 차감 방식 운영. 대량 구매 시 크레딧 보너스 제공. 영업·파트너사 대상 운영.
선납 락인 프로모션 — 영업 무기
6개월 이상 선납 시 혜택
→업종 독점 프리미엄 일부 할인 (전액 면제 불가)
→추가 업종 독점 할인

핵심 원칙
"독점 프리미엄 전액 면제 없음"
전액 면제 시 독점의 가격 권위가 무너집니다. 장기 선납 고객에게 더 많은 혜택이 쌓이는 구조로 설계. 할인이 아닌 "장기일수록 더 유리해지는 구조"로 영업합니다.
결제 구조 한 줄 요약: 첫 계약은 선납으로 안전하게, 갱신부터는 인보이스로 편리하게, 장기 선납 고객에게만 혜택을 더 크게.
