# CLYNK — Blueprint Portal v9.0

KDDC 기반 AI 셀럽 광고 매칭 플랫폼 **CLYNK**의 서비스 덱입니다.  
단일 `index.html` 파일로 구성되어 있으며, Vercel을 통해 정적 사이트로 배포됩니다.

## 관리자 기능

로그인 후 우측 하단 툴바 및 사이드바에서 사용 가능합니다.

| 기능 | 설명 |
|------|------|
| **인라인 편집** | 본문·카드·수치 등 텍스트 직접 수정 |
| **섹션 ON/OFF** | 섹션별 노출 여부 토글 (열람자에게도 적용) |
| **HTML 내보내기** | 현재 상태를 독립 실행형 HTML 파일로 다운로드 |

---

## 덱 구성

| # | 섹션 | 내용 요약 |
|---|------|----------|
| 00 | **핵심 강점** | KDDC DB 기반 즉시 실행 / 검색·매칭 패러다임 / 업종 독점 자동 보호 |
| 01 | **시장 구조** | 한국 광고 시장 변화와 CLYNK의 포지션 |
| 02 | **검색·매칭** | AI 기반 모델 추천·ROI 역산·업종 공백 실시간 표시 |
| 03 | **수익 구조** | 에스크로 정산, 수수료 체계, 머니 플로우 |
| 04 | **모델 구독** | 구독 플랜·패키지 구성 |
| 05 | **서비스 플로우** | 계약~제작~정산 전체 프로세스 |
| 06 | **제작 서비스** | 이미지·영상(6초·15초·30초·60초) AI 제작 |
| 07 | **5개년 재무** | 인터랙티브 재무 시뮬레이터 포함 |

---

## 기술 스택

- **순수 HTML / CSS / Vanilla JS** — 프레임워크·빌드 도구 없음
- **Pretendard** 폰트 (CDN)
- 인증·설정 저장: 브라우저 `localStorage`
- 스크롤 애니메이션: `IntersectionObserver`
